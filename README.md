# PVZ-schedule

Прототип веб-приложения для учета графика сотрудников и расчета зарплаты.

## Возможности
- Добавление и удаление сотрудников со ставкой за час.
- Календарь по месяцам с выбором типа смены на каждый день.
- Поддержка смен: выходной, 4ч, 8ч, 12ч, а также пользовательские часы.
- Автоматический расчет часов, количества смен и суммы к выплате за месяц.
- Экспорт отчета в CSV.
- Сохранение данных в `localStorage`.

## Локальный запуск
Откройте `index.html` в браузере или запустите любой статический сервер, например:

```bash
python -m http.server 8080
```

## Деплой на Vercel
1. Установите CLI: `npm i -g vercel`
2. Выполните в корне проекта: `vercel`
3. Для production: `vercel --prod`

Для текущего проекта дополнительная серверная конфигурация не требуется (статический сайт).
