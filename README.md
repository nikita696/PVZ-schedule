# PVZ-schedule

Веб-приложение для ведения графика смен и расчета выплат.

## Что реализовано
- График на месяц в табличном формате (день, день недели, сотрудники).
- Статусы дня: `Рабочий`, `Выходной`, `Отпуск`, `Больничный`, `Невыход`.
- Расчет зарплаты **только по рабочим сменам**.
- Добавление/удаление сотрудников.
- Изменение ставки каждого сотрудника прямо в блоке зарплаты.
- Кнопка «Выдать выплату» (фиксирует выплату за текущий месяц).
- Экспорт выплат в CSV.
- Сохранение данных в `localStorage`.

## Запуск
Откройте `index.html` в браузере.
