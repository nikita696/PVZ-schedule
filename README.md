# PVZ-schedule

Веб-приложение для ведения графика смен и расчета выплат в стиле табличного макета.

## Что реализовано
- Навигация по месяцам (назад/вперед/сегодня).
- Табличный график: `День`, `День недели` и колонки сотрудников.
- Статусы на каждый день: `Рабочий`, `Выходной`, `Отпуск`, `Больничный`, `Невыход`.
- Цвета статусов прямо в `<select>`.
- Правый блок «Зарплата и выплаты» с подсчетом смен, начислений, выплаченных сумм и остатка к выдаче.
- Кнопка «Выдать выплату» (фиксирует текущий остаток как выплаченный).
- Экспорт таблицы выплат в CSV (Excel-compatible).
- Автосохранение в `localStorage`.

## Запуск
Достаточно открыть `index.html` в браузере.
